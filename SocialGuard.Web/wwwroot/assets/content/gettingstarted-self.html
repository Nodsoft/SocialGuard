Integrating SocialGuard

You can create your own implementation of SocialGuard, and integrate it with your own Discord Bot.

Using our API

All our API Requests are documented thorougly on our Swagger, to help you with Development of SocialGuard functionality.

List User Records

Listing user records is the simplest function possible on our API :
https://socialguard.net/api/v3/User/list

Fetch a User Record

You can fetch a User's record by using this API Request : https://socialguard.net/api/v3/User/{ids}
(Where {ids} is replaced by a list of Discord Snowflakes/IDs, comma-delimited : ID1,ID2,ID3)

Taking Snowflake 611621392286482441 as an example, you call this endpoint to fetch its record :
https://socialguard.net/api/v3/User/611621392286482441
