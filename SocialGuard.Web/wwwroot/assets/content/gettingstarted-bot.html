Using a Bot

SocialGuard is available on several bots, which you can use.

Yume-Chan

Open-Source, endorsed

Invite to server
Documentation

Yume-Chan is our prefered solution, as it follows closely the development cycle of the SocialGuard API. This bot is holder of many plugins, including the NSYS SocialGuard for Yume-Chan plugin.

To use Yume-Chan to its fullest, it is recommended to grant it Administrator (0x8) permission. However, the SocialGuard plugin uses only a subset of these. You can also grant only the "Ban Members" (0x4) permission, and basic functionality should still work fine.
